Key constraints and known bugs to fix:
- Do NOT change desktop layout — all fixes are inside media queries
- The detail page media query `section[style*="grid-template-columns: '1fr 1fr'"]` is BROKEN — CSS attribute selector with contains (`*=`) on a React inline style will not reliably match because React serializes style objects as `grid-template-columns:1fr 1fr` (no quotes around value). This broken selector is why the 2-column intro does not collapse on mobile today.
- The 3-column gallery in [slug]/page.tsx has no mobile override at all — must add one.
- Use className-based fixes: add a className to each element, then target that className in the embedded media query.

From src/app/projects/[slug]/page.tsx (three broken/missing areas):

Area 1 — Project intro section (line 81):
```jsx
<section style={{ padding: '80px 48px', display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '80px', ... }}>
```
Fix: Add className="project-intro" to the section element and add to embedded style:
```css
@media (max-width: 768px) {
  .project-intro { grid-template-columns: 1fr !important; gap: 40px !important; padding-top: 48px !important; padding-bottom: 48px !important; }
}
```

Area 2 — Photo gallery inner div (line ~111):
```jsx
<div style={{ display: 'grid', gridTemplateColumns: 'repeat(3, 1fr)', gap: '1.5px', ... }}>
```
Fix: Add className="project-gallery-grid" to this div and add to embedded style:
```css
@media (max-width: 768px) {
  .project-gallery-grid { grid-template-columns: 1fr !important; }
}
```

Area 3 — Next project Link (line ~132):
The Link has display: flex, alignItems: center, justifyContent: space-between. At 360px the title can push the "View" span off. Fix: add className="next-project-link" and add:
```css
@media (max-width: 768px) {
  .next-project-link { flex-direction: column !important; align-items: flex-start !important; gap: 16px !important; }
}
```

Current embedded style to keep + extend (replace broken selector):
```css

  /* broken: section[style*="grid-template-columns: '1fr 1fr'"] — remove this */
}
```
